# Skribbl Clone

Live URL:
https://skribbl-clone-dun.vercel.app/

## Features
- Create Room
- Join Room
- Real-time Drawing
- Real-time Chat
- Word Selection
- Scoring System
- Multi-round Gameplay
- Winner Screen

## Tech Stack
Frontend:
- React
- Vite
- Socket.IO Client

Backend:
- Node.js
- Express
- Socket.IO

## Installation

Client:
npm install
npm run dev

Server:
npm install
npm run dev
